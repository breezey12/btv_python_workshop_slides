String Theory Practice

Remember, Python recognizes a string when it sees 'single' or "double" quotes


phrase = 'Hello, World'

my_string = "We're #1"

string_number = "12345"

Bad Idea: Doing Math with Strings

>>> my_number = "12"
>>> print my_number * 3
121212
>>>

121212 is cute, but probably not very useful.

Good Idea: Convert to Integers First, and then Do Math

>>> my_number = int(12)  
>>> my_number * 3
36
>>>

Or use float() to add a decimal point:

>>> my_number = float(12)
>>> my_number * 3
36.0
>>>

Important "String" Instruments:

Mary Poppins using python's len()

>>> len("Supercalifragilisticexpialidocious")
34
>>>

Find a string within a string with find():

>>> phrase = "Great, 23 encyclopedias of text typed out for us when python is the only topic we care about."
>>> print phrase.find("python")
54
>>>

Users Input

How to take in and pick apart a users input in your python program


User Input Example

Use the raw_input() function to take in text from a user


birthplace = raw_input("Hey, where were you born?")
print "You were born in: ", birthplace

Save as userinput.py and run this on your command line in Python by typing:

>>> python userinput.py

This is what your program looks like:


Brendans-MacBook-Air:workshop brendanreese$ python userinput.py 
Hey, where were you born?Hanover, NH
You were born in:  Hanover, NH

Lists

Lists are a data type that contain items


>>> desserts = ["ice cream", "cookies", "brownies", "tiramisu"]
>>> scores = [10, 8, 9, -2, 9]

List Tricks

How to get a single item out of a list:


>>> desserts = ["ice cream", "cookies", "brownies", "tiramisu"]
>>> print desserts[2]
brownies
>>>

How to get multiple items out of a list:


>>> desserts = ["ice cream", "cookies", "brownies", "tiramisu"]
>>> print desserts[1:3]
['cookies', 'brownies']
>>>

How to create an empty list and start adding to it:


>>> animals = []   # create an empty list with []
>>> animals.append("lions")
>>> animals.append("tigers")
>>> animals.append("bears")
>>> print animals
['lions', 'tigers', 'bears']
